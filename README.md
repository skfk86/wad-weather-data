# wad-weather-data

بيانات طقس السودان — تُحدَّث كل 30 دقيقة تلقائياً.

## روابط البيانات

| الملف | الوصف |
|---|---|
| `/weather/snapshot.json` | بيانات جميع المدن الحالية |
| `/weather/alerts.json` | التنبيهات النشطة |
| `/weather/meta.json` | وقت آخر تحديث |
| `/weather/cities/{city}.json` | بيانات مدينة محددة |

تطبيق: **طقس السودان — ود السحاب**
